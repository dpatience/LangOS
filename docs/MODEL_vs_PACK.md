# Models vs Packs

| | `packs/` | `models/` |
|---|----------|-----------|
| **What** | Language knowledge (rules, grammar, templates) | Trained statistical weights |
| **Created by** | Hand-authored + `mix patience train` (lexicon) | `mix patience train` only |
| **Editable** | Yes (JSON/YAML) | No — regenerate |
| **Required to run** | Yes | No (rules work without models) |
| **Improves** | Pattern matching, syntax, generation | Free-form intent classification |

## `packs/` — language packs

One folder per language. Defines **how LangOS speaks and parses** that language.

```
packs/de/
├── manifest.json
├── grammar.json              # parts of speech, morphology, realizer rules
├── patterns/commands.json    # regex patterns, verb_map, pronoun_map
├── lexicon.json              # generated by train (word → vocab ID)
├── templates/express/        # missing_fields, success, error, ...
└── tests/golden.jsonl        # test cases
```

Install: `mix patience install language de`

## `models/` — trained stat engine

```
models/de/intent.json         # Naive Bayes intent classifier
```

Build: `mix patience train --lang de`  
Used by: `LangOS.Engine.Stat` (loads `Model.intent("de")`)

## Flow

```
Author pack (grammar, patterns, golden tests)
        │
        ▼
mix patience install language xx     ← runtime: rules + syntax work
        │
        ▼
mix patience train --lang xx         ← offline: lexicon + intent.json
        │
        ▼
Stat engine uses models/xx/intent.json for free-form text
```

## When to touch which

| Task | Action |
|------|--------|
| Fix German grammar | Edit `packs/de/grammar.json` |
| Add command pattern | Edit `packs/de/patterns/commands.json` |
| Improve free-form intent | Add golden tests, then `mix patience train --lang de` |
| Add new language | See [LANGUAGE_PACK_GUIDE.md](./LANGUAGE_PACK_GUIDE.md) |
